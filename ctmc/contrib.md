# 贡献

## 为此文档贡献内容

点击右上角 <TkIcon icon="mdi:github"/> 按钮跳转至我们的 Github 仓库，或者[传送门](https://github.com/AstralFlare-owo/TZ-Site/fork)直接跳转到 Fork 页面，

Fork 一下仓库，在 `/ctmc/` 目录下创建/修改内容，然后提交 Pull Requests，等待审核。提交 PR 时请务必允许维护者修改内容。

若创建了新的页面，请在 `/.vitepress/config.mts` 的 `defineConfig.themeConfig.sidebar` 中添加页面链接。

:::details 新页面模板
```markdown
---
title: 标题
article: false
author:
  name: CTMC Community
---

# 标题

正文
```

请修改模板中的两处标题和你的正文内容。
:::
